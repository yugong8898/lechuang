# USER_GUIDE.md — 使用说明

## 一、功能概览

| 功能 | 说明 |
|------|------|
| CSV 数据导入 | 上传 `app_roi_data.csv`，自动存入数据库 |
| 多维度筛选 | 按渠道、出价类型、国家、APP 筛选 |
| ROI 趋势图 | 8条时间维度趋势线 + 100% 回本参考线 |
| 移动平均切换 | 7日移动平均 / 原始数据 |
| Y轴刻度切换 | 线性 / 对数 |
| 图例显隐 | 点击图例隐藏/显示对应曲线 |
| 预测虚线 | 日期不足数据以虚线展示 |
| 悬停详情 | 悬停显示当日所有维度 ROI |

---

## 二、数据导入

### 方式一：页面导入（推荐）

1. 打开 `http://localhost:3000`
2. 点击右上角 **「导入 CSV」**
3. 选择 `app_roi_data.csv`
4. 等待完成提示

### 方式二：命令行

```bash
cd backend
npm run import
# 或指定路径
npx ts-node src/scripts/importCsv.ts /path/to/data.csv
```

### CSV 列格式

| 列名 | 示例 |
|------|------|
| 日期 | 2025-04-13(日) |
| app | App-1 |
| 出价类型 | CPI |
| 国家地区 | 美国 |
| 应用安装.总次数 | 140 |
| 当日ROI ~ 90日ROI | 6.79% |

---

## 三、筛选器

**第一行下拉（实时生效）：**
- 用户安装渠道：Apple / 全部
- 出价类型：CPI / 全部
- 国家地区：美国 / 英国 / 全部
- APP：App-1 ~ App-5 / 全部

**第二行单选：**
- 数据显示模式：`显示移动平均值`（7日滑动）/ `显示原始数据`
- Y轴刻度：`线性刻度` / `对数刻度`（适合查看跨量级数据）

---

## 四、图表说明

### 数据线颜色

| 颜色 | 维度 |
|------|------|
| 灰色 | 当日 ROI |
| 天蓝 | 1日累计 |
| 绿色 | 3日累计 |
| 紫色 | 7日累计 |
| 橙色 | 14日累计 |
| 粉色 | 30日累计 |
| 黄色 | 60日累计 |
| 亮绿 | 90日累计 |
| 红色虚横线 | 100% 回本线 |

### 虚线含义

曲线末端的**虚线段**表示「日期不足」——该日期距截止日（2025-07-12）不足对应统计周期，数据不具参考意义。

### 图例交互

点击底部图例文字可切换对应曲线的显示/隐藏状态。

### 悬停 Tooltip

鼠标移入图表区域，浮窗显示当日所有维度的精确 ROI 百分比值。
